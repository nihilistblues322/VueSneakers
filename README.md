


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

⚙️ Стэк технологий:

    Vue 3
    Vue Router
    TailwindCSS
    Axios
    @formkit/auto-animate
    Composition API / Options API
    ESLint
    Prettier
